# Diva Nails

Aplicación de escritorio para la gestión de un spa de uñas. Permite registrar citas, clientes, servicios, manicuristas y generar reportes de caja.

## Tecnologías

- Flutter (Desktop - Windows)
- SQLite (sqflite_common_ffi)
- SharedPreferences
- FilePicker
- Excel export (CSV)

## Funcionalidades

- **Registro**: Citas diarias con hora, cliente, servicios, manicurista, pagos
- **Clientes**: CRUD con búsqueda, cumpleaños, ubicación
- **Servicios**: CRUD de servicios con precios
- **Manicuristas**: CRUD con porcentaje de ganancia
- **Caja**: Reportes semanales o por rango de fechas, exportación a Excel
- **Ajustes**: Modo oscuro, color de tema, rutas de base de datos y exportación
